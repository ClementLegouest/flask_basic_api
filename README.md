# Flask_basic_api

## Scope

This is a minimal API using the flask micrfrmework
This is meant to run on Python 3

## Run the API

### Install PIP

[PIP](https://pip.pypa.io/en/stable/installing/)

### Create a folder for your project

$ mkdir myproject

### Create a venv for your project

$ cd myproject
$ python3 -m venv venv

### Activate the venv

$ . venv/bin/activate

### Install Flask

$ pip install Flask

### Run the API

$ python3 main.py
